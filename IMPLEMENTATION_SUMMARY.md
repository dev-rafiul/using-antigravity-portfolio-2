## 🚀 How to Use

### 1. Run the Project
```bash
npm install
npm run dev
```

### 2. Customize Content
- Update personal info in `Hero.jsx`
- Add your photo (place in `public/` folder)
- Add your resume as `public/resume.pdf`
- Update About Me in `About.jsx`
- Update skills in `Skills.jsx`
- Update projects in `Projects.jsx`
- Update experience in `Experience.jsx`
- Update education in `Education.jsx`
- Update contact info in `Contact.jsx`

### 3. Deploy
```bash
npm run build
# Deploy 'dist' folder to Vercel/Netlify
```
